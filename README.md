# 🚀 PC-Booster: Ultra Windows & Gaming Performance Optimizer

A lightweight, fully transparent, and open-source `.bat` script designed to eliminate input lag, lower network ping, and maximize FPS in competitive games like *Valorant, Counter-Strike 2, Assetto Corsa, and Apex Legends*. 🎮✨

Most "PC Boosters" on the internet are bloated, closed-source `.exe` files that track your data or contain malware. **PC-Booster** is completely open-source—you can inspect every single line of code before running it. Clean, safe, and effective.

---

## 🛠️ What It Does (Technical Breakdown)

This script automates advanced Windows tweaks that usually take hours to configure manually. Here is exactly what happens under the hood:

*   **RAM & Cache Purge:** Flushes standby memory and clears system temporary files (`temp`, `%temp%`, `prefetch`) to free up system resources.
*   **Input Lag Reduction:** Tweaks the Windows registry to optimize keyboard and mouse responsiveness, ensuring your shots register instantly.
*   **Network & Ping Optimization:** Disables Nagle's Algorithm (TCP No Delay), flushes DNS cache, and optimizes network throttling to minimize packet loss and in-game ping.
*   **CPU & Power Management:** Unlocks the hidden **"Ultimate Performance"** power plan in Windows to ensure your hardware runs at maximum clock speed without artificial throttling.
*   **Background Services Debloat:** Disables non-essential Windows telemetry, error reporting, and heavy background tracking services that hog CPU cycles while gaming.

## 🚀 How to Download & Run (10-Second Setup)

Since this script modifies advanced system and registry settings, it **must be run as an Administrator**.

1.  Click the green **"Code"** button at the top of this page and select **"Download ZIP"**.
2.  Extract the ZIP file to your desktop.
3.  Right-click on `PC-Booster.bat` and select **"Run as administrator"**.
4.  Follow the on-screen prompts, restart your PC once the script finishes, and enjoy the performance boost! 🏁

> ⚠️ **Disclaimer:** *It is highly recommended to create a Windows Restore Point before running any system optimizer. Use at your own risk.*
